Read `AGENTS.md`, then load `CONTEXT.md` (or the individual `knowledge/*.md` and
`tools/*/` files) as background for this repo. This is an **AI context repo**
distilling years of one operator's EVE Online knowledge — not scoped to any
single tool. Today it covers the EVE ESI API, the EVE SDE, operator
methodologies, and the third-party tools an EVE Online operator running 60+
characters uses on top of them — eveforge.org (EVEFORGE) currently, more planned
under `tools/`. It is structural/reference knowledge only — never assert prices
or other live game data from memory; point to the relevant tool/ESI/in-game for
that. The operator is an EVE expert — skip game fundamentals.
